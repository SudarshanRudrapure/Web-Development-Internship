Interactive Web Page Mini Task – Explanation

This webpage demonstrates a simple interactive feature using JavaScript. The page contains a button and a paragraph element that allows the user to interact with the webpage.

When the user clicks the button, a JavaScript function is triggered. This function changes the text displayed on the page, showing how JavaScript can respond to user actions.

The program uses the getElementById() method to select the paragraph element and update its content dynamically. This demonstrates how JavaScript can modify HTML elements in real time.

JavaScript is commonly used to handle events such as button clicks, keyboard input, and form submissions. By combining HTML for structure, CSS for styling, and JavaScript for functionality, developers can create dynamic and interactive web applications.
